# Frontend Playground

A collection of small frontend projects built using HTML, CSS, and JavaScript. This repository serves as a practice space for experimenting with user interface designs, animations, and basic web development concepts.

## About

This repository contains mini-projects created while learning and exploring frontend development. Each project focuses on a specific concept such as DOM manipulation, CSS animations, hover effects, or interactive UI components.

The goal of this repository is to improve practical web development skills through hands-on experimentation.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Repository Structure

```text
Frontend-Playground
│
├── Animated-Cards
├── Light-Bulb-Toggle
└── Square-Hover-Effect
```

## Projects Included

### 1. Animated Cards

A responsive card design project featuring hover animations and modern UI styling. It demonstrates the use of CSS transitions, transforms, and interactive design principles.

### 2. Light Bulb Toggle

A JavaScript project that simulates turning a light bulb on and off. It demonstrates basic DOM manipulation, event handling, and dynamic content updates.

### 3. Square Hover Effect

A simple project demonstrating CSS hover effects and transitions. When the user hovers over a square element, visual changes are triggered to create an interactive experience.

## Learning Objectives

Through these projects, I practiced:

* HTML page structure
* CSS styling and layouts
* CSS transitions and animations
* JavaScript event handling
* DOM manipulation
* Responsive design fundamentals
* Frontend project organization
